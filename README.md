# Presentación proyecto

**Integrantes:**
* Antonia Araya
* Emilia Bertrán
* Anghy Gil

El proyecto propone una investigación de datos sobre la participación de Chile en el certamen Miss Universo, específicamente de aquellas candidatas que han logrado clasificar en el Top 12. El foco principal no es la estética individual, sino la búsqueda de patrones estructurales que vinculen el éxito en el concurso con el origen geográfico y socioeconómico de las representantes.
