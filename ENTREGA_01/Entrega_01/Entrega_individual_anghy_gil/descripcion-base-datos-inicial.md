Base de datos: Indicadores socioeconómicos por territorio
Descripción general
Esta base de datos reúne indicadores socioeconómicos de las regiones y comunas de Chile, con el fin de analizar su posible relación con el desempeño de las candidatas en Miss Universo.


Fuente de datos
Instituto Nacional de Estadísticas (INE)
Encuesta CASEN
Reportes oficiales del gobierno de Chile


Contenido
La base incluirá las siguientes variables:
region
comuna (si es posible)
ingreso_promedio
nivel_educacion
indice_desarrollo


Metodología de recolección
Los datos serán obtenidos desde fuentes oficiales y consolidados en un único archivo.
Etapas:
Identificar indicadores socioeconómicos relevantes
Descargar bases de datos oficiales
Seleccionar variables necesarias
Estandarizar nombres de regiones y comunas
Integrar la información en formato CSV


Estructura del archivo
Formato CSV:
region,comuna,ingreso_promedio,nivel_educacion,indice_desarrollo


Pertinencia
Esta base es clave para evaluar la hipótesis del proyecto, ya que permite analizar si existe una relación entre el nivel socioeconómico del territorio de origen y el éxito en el certamen.
