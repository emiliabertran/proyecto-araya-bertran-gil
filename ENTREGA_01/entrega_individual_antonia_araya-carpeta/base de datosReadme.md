# Base de datos: Origen territorial de las candidatas

## Descripción general
Esta base de datos busca identificar la región y comuna de origen de las representantes chilenas en Miss Universo, con el fin de analizar la distribución territorial de las candidatas.

## Fuente de datos
- Perfiles en medios de comunicación
- Entrevistas
- Biografías públicas
- Redes sociales verificadas

## Contenido
La base incluirá las siguientes variables:
- nombre
- región de origen
- comuna de origen (si se cuenta con la información)

## Metodología de recolección
La información será recopilada manualmente mediante búsqueda individual por Miss.

### Etapas:
1. Buscar información biográfica de cada candidata  
2. Identificar su lugar de origen (región y comuna)  
3. Verificar la información en más de una fuente  
4. En caso de no encontrar datos exactos de comuna, registrar la región como mínimo  

## Estructura del archivo
Formato CSV: nombre,region_origen,comuna_origen


## Pertinencia
Esta base permite analizar la distribución geográfica de las candidatas y es clave para posteriormente poder cruzar los datos con variables socioeconómicas.
